Создать класс, который реализует интерфейс java.Класс должен разбирать текстовый файл и возвращать предложения из файла. Метод iterator данного класса должен возвращать объект итератор - экземпляр внутреннего класса. Метод Iterator#remove должен бросать исключение UnsupportedOperationException.

Не допускается использовать существующие реализации итераторов из контейнерных классов!

Замечание. Напишите регулярное выражение, которое "вырезает" предложения из текста, далее используйте объект Matcher при реализации методов интерфейса Iterator.
